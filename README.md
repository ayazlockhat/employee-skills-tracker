# employee-skills-tracker

# Tech Stack

## **Frontend**

* **Next.js 15** (JavaScript)
* **App Router**
* **Tailwind CSS**
* **shadcn/ui** (UI components)
* **react-chartjs-2** (charts)
* **Fetch API** for backend communication

## **Backend**

* **Python 3 + Flask**
* **Flask-CORS** (handling cross-origin requests)
* **PyMongo** (MongoDB driver)
* **pandas + openpyxl** (Excel import/export)
* Layered architecture:

  * `routes/`
  * `services/`
  * `repositories/`
  * `utils/`

## **Database**

* **MongoDB Atlas**
  Cloud-hosted MongoDB cluster for employee data storage.

## **Deployment**

* **Frontend:** Vercel
* **Backend:** Render
* **Database:** MongoDB Atlas

## **CI/CD**

* **GitHub Actions** for automated deployment of both frontend and backend.
